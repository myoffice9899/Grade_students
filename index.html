Tab 1: เจอ bug กดปุ่ม แล้วไม่คำนวณ

function calculate() {
    // ไม่มีโค้ดสำหรับการคำนวณ
}

ตัวอย่างโปรแกรม

โค้ดไม่มีฟังก์ชันสำหรับการคำนวณ ทำให้กดปุ่มแล้วไม่เกิดอะไรขึ้น

Tab 2: คำนวณยอด 10000 ผิดพลาด โดยยังได้รับเป็นโปรโมชั่น 2 %

function calculate(amount) {
    if (amount >= 10000) {
        return amount * 0.02; // ส่วนลดผิดพลาด
    }
}

ตัวอย่างโปรแกรม

โค้ดกำหนดส่วนลดผิดพลาดสำหรับยอดซื้อ 10,000 บาท ซึ่งควรได้รับส่วนลด 3.5%

Tab 3: กดคำนวณได้ แต่ไม่แสดง ยอดที่ต้องจ่ายทั้งหมด

function calculate(amount) {
    const discount = amount * 0.03;
    // ไม่มีการแสดงผลยอดที่ต้องจ่าย
}

ตัวอย่างโปรแกรม

โค้ดไม่ได้แสดงผลรวมหลังหักส่วนลด ทำให้ผู้ใช้ไม่ทราบยอดที่ต้องจ่าย

Tab 4: กรอกน้อยกว่า 1000 เจอ error ระบบไม่เข้าร่วมรายการ

function calculate(amount) {
    if (amount < 1000) {
        throw new Error('ระบบไม่เข้าร่วมรายการ');
    }
}

ตัวอย่างโปรแกรม

โค้ดแสดงข้อผิดพลาดเมื่อยอดซื้อน้อยกว่า 1,000 บาท ซึ่งเป็นการปฏิเสธไม่เข้าร่วมโปรโมชั่น

Tab 5: แสดงผล ระบบทำงานผิดพลาด

function calculate() {
    throw new Error('ระบบทำงานผิดพลาด');
}

ตัวอย่างโปรแกรม

โค้ดแสดงข้อความทั่วไปว่า "ระบบทำงานผิดพลาด" โดยไม่ได้ระบุสาเหตุที่แท้จริง
